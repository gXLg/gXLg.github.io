чонкер BOT Now
Использование данных
Чонкер просматривает твои сообщения, чтобы отвечать на команды. Чонкер не сохраняет содержание твоих сообщений никаким образом. При использовании команды `>_do` код, набранный тобою, будет сохранён для выполнения и удалён после двух часов неактивности. Также для выполнения кода создаётся приватная среда с названием твоего пользовательского id.
Используя чонкера, ты соглашаешься с вышеназванными условиями.

Для работы чонкера требуется сохранение id сервера в базе данных. При покидании сервера данные сервера не удаляются. Однако можно запросить их удаление на сервере поддержки.
Приглашая бота на твой сервер, ты соглашаешься с вышеназваннами условиями.
